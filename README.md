## 🧭 What this app does

snapshot_poll is a lightweight polling tool for DAO groups and other online teams. It helps people share a signal without a heavy vote process.

Use it when you want quick input from a group, such as:

- picking between two options
- checking support for a proposal
- gathering sentiment before action
- coordinating decisions across a web3 group

It is built to stay simple. You can use it without a long setup flow or extra tools.
